HTML (HyperText Markup Language) is the standard language used to create web pages. It uses various tags to structure and format text content.

1. Headings

Headings are defined using tags from <h1> to <h6>. <h1> is the largest heading and <h6> is the smallest.

This is an H2 heading

This is an H5 heading
This is an H6 heading

As you can see, each heading level decreases in size and importance.

2. Paragraphs

A paragraph is defined using the <p> tag. Paragraphs are used to display blocks of text on a web page.

Example paragraph: HTML paragraphs automatically have some spacing between them to improve readability.

3. Text Formatting Tags

HTML provides several tags to format text. Some commonly used formatting tags include:

Bold text: using <b> or <strong>
Italic text: using <i> or <em>
Underlined text: using <u>
Highlighted text: using <mark>
Small text: using <small>
