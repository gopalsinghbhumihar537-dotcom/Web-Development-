Student Information Form

Student Information











































Text Inputs: Collect short text like Name, Email, Mobile, or Address. Specialized types like email, tel, and date help with validation.

Radio Buttons: Allow the user to select only one option from a set. Example: Gender.

Checkboxes: Allow the user to select multiple options. Example: Skills or Hobbies.

Select Menus: Provide a dropdown list for choosing one option. Example: Branch or Year.

Submit Button: Sends the form data to the server.

Reset Button: Clears all input fields and restores default values.
